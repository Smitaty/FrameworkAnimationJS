Animation

1. Description

Le fichier Animation.jar est un fichier archive java qui contient une Applet permettant de réaliser de petites animations à des fins de démonstrations pédagogiques.

Voici un petit exemple de ce qui peut être réalisé rapidement.

2. Utilisation

Télécharger le fichier Animation.jar et inclure le code source suivant au sein de la page XHTML : 


    <applet codebase="." 

     code="AnimationApplet.class"

     archive="Animation.jar"

     alt="Animation Applet"

     width="640" height="480">

    <param name="config" value="animation.xml" />

    </applet>

Vous pouvez modifier les paramètres suivants :

  • width : largeur en pixels de l'espace occupé par l'Applet
  • height : hauteur en pixels de l'espace occupé par l'Applet
  • animation.xml : il s'agit du fichier qui décrit l'animation

3. Fichier de description de l'animation

Ce fichier est au format XML, la DTD qui lui correspond est animation.dtd

Le fichier XML se compose de 3 parties :

  • background : permet de spécifier une image de fond
  • objects : permet de décrire les différents objets de l'animation
  • programs : permet de décrire les actions associées aux objets

3.1 Les objets

Les objets sont définis par :

  • un identifiant unique donné par l'utilisateur
  • des coordonnées x,y qui indiquent leur position
  • des couleurs de fond, d'affichage et éventuellement de cadre
  • un état représenté par une chaîne de caractères
  • un indicateur de la couche d'affichage de l'objet
d'autres attributs existent mais ne sont pour le moment pas utilisés :
  • angle de rotation de l'objet

On distingue 5 objets de base :

  • object_text pour le texte
  • object_box pour un rectangle
  • object_image pour une image (sprite)
  • object_polygon pour une forme définie par l'utilisateur
  • object_ellipse ellipse

Note : l'utilisateur peut incorporer ses propres objets (cf. créer ses propres objets)

3.1.1 object_text

Les attributs pour un texte sont les suivants :

  • text : le texte, ou le caractère @ indique un retour chariot
  • font : la fonte décrite au format AWT de Java
  • fgcolor : couleur du texte format RGB
  • bgcolor : couleur du fond
  • bocolor : couleur du bord du cadre englobant le texte
  • border : nombre de pixels entre texte et cadre du texte
  • transparency : booléen (true|false), indique si on utilise la couleur de fond
  • layer : couche de superposition des objets, par défaut à 0. Un objet qui possède une valeur de couche supérieure à un autre, sera dessiné après cet objet

Exemple : texte sur 2 lignes en Courier normal 12 points, couleur de fond noir, couleur de texte blanc


<object_text

  text="click on text@to start"

  font="Courier,PLAIN,12"

  fgcolor="255,255,255" 

  bgcolor="0,0,0"

  border=2

  transparency="false"

  layer="2" >Mon_Texte<object_text>

3.1.2 object_image

Les attributs pour une image sont les suivants :

  • image : chemin jusqu'à l'image
  • fgcolor : couleur du texte format RGB
  • bgcolor : couleur du fond
  • layer : couche de superposition des objets, par défaut à 0. Un objet qui possède une valeur de couche supérieure à un autre, sera dessiné après cet objet

Exemple :


<object_image

  image="space_ship.jpg"

  layer="4" >Mon_Vaisseau<object_image>

3.1.3 object_box

Les attributs pour une boîte sont les suivants :

  • width : largeur en pixels
  • height : hauteur en pixels
  • fgcolor : couleur du texte format RGB
  • bgcolor : couleur du fond
  • bocolor : couleur du bord du cadre englobant le texte
  • layer : couche de superposition des objets, par défaut à 0. Un objet qui possède une valeur de couche supérieure à un autre, sera dessiné après cet objet

Exemple :


<object_box

  width="200"

  height="100"

  fgcolor="255,255,255" 

  bgcolor="0,0,0"

  layer="1" >Ma_Boite<object_box>

3.1.4 object_polygon

Les attributs pour un polygone sont les suivants :

  • coord_x : coordonnées sur l'axe des X
  • coord_y : coordonnées sur l'axe des Y
  • fgcolor : couleur du texte format RGB
  • bgcolor : couleur du fond
  • layer : couche de superposition des objets, par défaut à 0. Un objet qui possède une valeur de couche supérieure à un autre, sera dessiné après cet objet

Exemple :


<object_polygon

  coord_x="10,10, 0,15,30,20,20,10"

  coord_y=" 0,10,10,20,10,10,0, 0"

  fgcolor="255,255,255" 

  bgcolor="0,0,0"

  layer="1" >Ma_Boite<object_polygon>

3.2 Les programmes

On dispose d'un ensemble d'instructions pour modifier la position ou les propriétés des objets :

3.2.1 Déplacement

Instruction Attributs Description
setx
  • x : abscisse
 modifie l'abscisse de l'objet
sety
  • y : ordonnée
 modifie l'odonnée de l'objet
setxy
  • x : abscisse
  • y : ordinnée
 modifie abscisse et ordonnée
moveto
  • x : abscisse
  • y : ordonnée
  • dx : incrément sur l'axe x
  • dy : incrément sur l'axe y
  • delay : délai de temporisation
 déplace l'objet jusqu'à la position donnée
down
  • y : longueur du déplacement
  • dy : incrément sur l'axe y
 déplace l'objet vers le bas de y pixels
up
  • y : longueur du déplacement
  • dy : incrément sur l'axe y
 déplace l'objet vers le haut de y pixels
left
  • x : longueur du déplacement
  • dx : incrément sur l'axe x
 déplace l'objet vers la gauche de x pixels
right
  • x : longueur du déplacement
  • dx : incrément sur l'axe x
 déplace l'objet vers la droite de x pixels

3.2.2 Temporisation

Instruction Attributs Description
sleep
  • value : délai de temporisation
 temporisation en nombre de cycles d'affichage
state
  • value : état
 passe l'objet courant dans l'état donné
wait
  • value : état
 attend que l'objet passe dans l'état indiqué
trigger
  • object : nom d'objet
  • value : état
 passe l'objet spécifié dans l'état donné

3.2.3 Affichage

Instruction Attributs Description
visible
  • value : true ou false
 apparition ou disparition de l'objet
blink
  • times : nombre de clignotements
  • délai : délai entre 2 clignotements
 faire clignoter l'objet

3.2.4 Répétition (boucle)

Instruction Attributs Description
label
  • value : identifiant
 définit une ligne de code ou l'on pourra revenir
goto
  • value : identifiant de label
 saute au label indiqué

3.2.5 Modification des propriétés

Instruction Attributs Description
setproperty
  • object : identifiant d'objet
  • property : propriété à modifier
  • value : valeur
 permet de modifier une propriété de l'objet comme la couleur de fond (bgcolor), la couleur d'affichage (fgcolor), la couleur du cadre (bocolor), le texte (text) pour un objet de type texte.

3.2.6 Divers

Instruction Attributs Description
click aucun attends que l'on clique sur l'objet
stop aucun arrête l'applet

Exemple :


<program assigned_to="Mon_Texte">

  <setxy x="100" y="100" />

  <visible value="true" />

  <right x="200" dx="1" />

</program>

4. Créer ses propres objets

Les objets externes (créés par l'utilisateur) sont intégrés dans la section objects grâce au tag object_extern qui prend 2 attributs :

  • class : le nom de la classe
  • constructor: une chaine de caratère qui comprend les paramètres pour la construction de l'objet

L'objet doit dériver de la classe AnimatedObject et founir les méthodes suivantes :

  • un constructeur : qui prend 3 paramètres
  • une méthode de dessin : pour l'affichage
  • un controleur : qui vérifie si on a cliqué sur l'objet

Exemple : on désire créer un objet cercle 


import java.applet.*;
import java.awt.*;

public class AnimatedCircle extends AnimatedObject {

    protected int rayon;
	
    public AnimatedCircle(Applet applet, String name, String cons) {
	super(applet,name);
	this.rayon=Integer.parseInt(cons);
	width=height=rayon;
    }
	
    public void draw(Graphics2D g2d) {
	g2d.translate(x, y);
	g2d.rotate(Math.toRadians(face_angle));
	g2d.setColor(bgcolor);
	g2d.drawRect(0, 0, width, height);
	g2d.fillOval(0, 0, rayon, rayon);
	g2d.setColor(fgcolor);
	g2d.drawOval(0,0, rayon, rayon);
	
     }

     public boolean isInside(int x, int y, Graphics2D g2d) {
	double xx = this.x + ((double)rayon/2) - (double)x;
	double yy = this.y + ((double)rayon/2) - (double)y;
	double r=Math.sqrt(xx*xx+yy*yy);
	return (r < (double)rayon);
     }
}

On met les fichiers AnimatedCircle.java et AnimatedCircle.class dans le même répertoire que Animation.jar.

On compile le fichier Java comme suit :


javac AnimatedCircle.java -jar Animation.jar

Enfin, pour créer un objet dans l'animation, on modifie le fichier de configuration XML comme suit :


<object>

  <object_extern class="AnimatedCircle" 

    constructor="50">MonCercle<object_extern>

</object>
