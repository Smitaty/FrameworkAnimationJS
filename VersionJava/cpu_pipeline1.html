Animation CPU sans Pipeline

Objectif

Montrer schématiquement comment sont traitées les instructions dans un microprocesseur sans pipeline.

Chaque instruction passe par les 5 phases de traitements :

  • Fetch : chargement depuis la mémoire
  • Decode : décodage
  • Load operand : chargement des opérandes
  • Execute : exécution
  • Write result : écriture du résultat
